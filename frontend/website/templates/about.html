About SNPglobal

{% endblock %} {% block content %}

SNPglobal is a web application that provides human population genetic information and summary statistics.


The database currently supports the single nucleotide polymorphism data of chromosome 22 for five human populations from the 1000 Genomes Project.


The genetic information stored in the SNPglobal database includes the alternate allele, reference allele and ancestral allele frequencies of each population. As well as the genotype frequencies of each population.


Using the data from the SNPglobal database, the user can select their summary statistic of interest for their genomic region and population of interest. This data is displayed in interactive plots that may be saved, with downloadable plotting data.
